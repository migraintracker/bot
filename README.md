# MigrainTracker Bot

Telegram-бот для отслеживания мигреней, анализа связи с погодой и магнитными бурями, и AI-прогнозирования приступов.

⚠️ **Не является медицинским изделием.** Все прогнозы - ознакомительные.

## Возможности

- **Запись приступов**: интенсивность, длительность, сторона, тип боли, аура, тошнота, свето/звукочувствительность, стресс, сон, триггеры, лекарства
- **Ежедневный чекин**: быстрая отметка «болит / не болит» для точной статистики
- **Погода**: текущая температура, давление, влажность, ветер, осадки, облачность - автоматически с Open-Meteo (Норвежский метеоинститут)
- **Космическая погода**: Kp-индекс магнитных бурь, скорость солнечного ветра, Bz-компонента - автоматически с NOAA (США)
- **AI-анализ**: нейросеть DeepSeek ищет закономерности между твоими приступами и погодой/Kp
- **Прогноз**: риск мигрени на 7 дней (низкий / средний / высокий / очень высокий)
- **Штормовые уведомления**: бот присылает предупреждение при сильных магнитных бурях (G2+)
- **Цикл**: отслеживание фаз менструального цикла (для женщин)
- **Экспорт**: скачай прогноз на неделю как .ics и импортируй в Google/Apple календарь
- **Напоминания**: ежедневный чекин в выбранное время (часовой пояс определяется по городу)
- **Приватность**: только обезличенные данные уходят в DeepSeek, пол - опционально

## Технологии

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.12 |
| Фреймворк | aiogram 3.x |
| База данных | PostgreSQL 16 (asyncpg) |
| Кеш/очереди | Redis + arq |
| ORM | SQLAlchemy |
| Контейнеризация | Docker Compose |
| Погода | Open-Meteo API (бесплатно, без ключа) |
| Косм. погода | NOAA SWPC API (бесплатно, без ключа) |
| AI | DeepSeek API |

## Быстрый старт

```bash
git clone https://github.com/migraintracker/bot.git
cd migraine-tracker-bot
cp .env.example .env
# Отредактируй .env - вставь свои ключи
docker compose up -d --build
```

Бот запущен на `bot:8080` (внутри сети), воркер стартует автоматически.

## Переменные окружения (.env)

| Переменная | Обязательно | Описание |
|-----------|-------------|----------|
| `BOT_TOKEN` | Да | Токен Telegram бота от @BotFather |
| `POSTGRES_PASSWORD` | Да | Пароль для PostgreSQL (смени дефолтный) |
| `DEEPSEEK_API_KEY` | Нет | Ключ DeepSeek (без него - эвристический fallback) |
| `WEATHERAPI_KEY` | Нет | Резервный источник погоды |
| `OPENWEATHERMAP_API_KEY` | Нет | Резервный источник погоды |

Остальные переменные имеют разумные значения по умолчанию.

## Источники данных

| Данные | Источник | Ключ | Бесплатно |
|--------|----------|------|-----------|
| Погода (основной) | [Open-Meteo](https://open-meteo.com) | Нет | Да |
| Погода (резерв) | WeatherAPI | Да | 1M запр./мес |
| Погода (резерв) | OpenWeatherMap | Да | 1000 запр./день |
| Kp-индекс, бури | [NOAA SWPC](https://swpc.noaa.gov) | Нет | Да |
| AI-анализ | [DeepSeek](https://deepseek.com) | Да | ~$0.14/1M токенов |

Open-Meteo агрегирует данные национальных метеослужб: ECMWF, DWD, MET Norway, Météo-France, NOAA GFS и других.

## Архитектура

```
docker-compose.yml
├── postgres:16       # Данные пользователей и погоды
├── redis:7           # Очередь задач arq
├── bot               # aiogram dispatcher (обработка команд)
└── worker            # arq worker (фоновые задачи)
    ├── sync_weather      - каждые 12ч (06:00, 18:00 UTC)
    ├── sync_space_weather - каждые 4ч (обновление Kp)
    ├── generate_predictions - раз в день (09:00 UTC)
    ├── send_reminders     - каждый час
    └── notify_kp_storm    - каждые 6ч (проверка бурь)
```

## Приватность

- **Telegram ID** - только для связи с тобой (обязательно)
- **Город** - для погоды и Kp по координатам (обязательно для прогнозов)
- **Пол / возраст** - опционально, повышают точность AI
- **Записи приступов** - только то, что ты сам ввёл
- В DeepSeek уходят только обезличенные данные (без имени, ID, точных координат)

Подробнее - в боте: кнопка **Помощь** → **Приватность и данные**.
